Installation

Step 1: Opening the box

To compile this source code, you will need to configure the makefile by edting paths and choosing preference compilers and linker.

Then make.

Step 2: Run

./ParEGO 3563563 f_oka2

runs ParEGO on the OKA2 fitness function with random seed 3563563.

Other fitness functions included in this code are called with the following parameters:

f_oka1 f_kno1 f_vlmop2 (also the default if no params are given) f_vlmop3 f_dtlz1a f_dtlz2a f_dtlz4a f_dtlz7a
